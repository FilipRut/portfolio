The site you're looking at right now

Custom-built portfolio with interactive 3D keychain, physics simulation, AI chat interface, and performance-optimized animations. Built with Astro, Three.js, and GSAP.

The Brief

Needed a portfolio that demonstrates both design thinking and technical craft. Most designer portfolios are static Webflow sites — I wanted something that feels alive, interactive, and memorable while remaining performant and accessible.

The Build

Custom Astro site with physics-based 3D keychain (Rapier + Three.js), AI-powered chat interface, scroll-driven animations via GSAP, and a content system for case studies. Every animation GPU-composited, lazy-loaded 3D scene, responsive from 320px to 4K.
